## 改动点列表

### 1. 修复第 78 行 - if 语句变量引用
**修改前：**
```bash
if [ $USB_COUNT -eq 0 ]; then
```

**修改后：**
```bash
if [ "$USB_COUNT" -eq 0 ]; then
```

**原因：** 防止 $USB_COUNT 为空时脚本崩溃

### 2. 修复第 79 行 - elif 语句变量引用
**修改前：**
```bash
elif [ $USB_COUNT -eq 1 ]; then
```

**修改后：**
```bash
elif [ "$USB_COUNT" -eq 1 ]; then
```

**原因：** 防止 $USB_COUNT 为空时脚本崩溃

### 3. 修复第 81 行 - 变量引用
**修改前：**
```bash
USB_PATH="${USB_LIST[$((CHOICE - 1))]}"
```

**修改后：**
```bash
USB_PATH="${USB_LIST[$(("$CHOICE" - 1))]}"
```

**原因：** 防止 $CHOICE 为空时出错
