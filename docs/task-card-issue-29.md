## 改动点列表

### 1. 修复第 106 行 - echo 语句路径引号
**位置：** start.bat 第 106 行
**修改前：**
```batch
echo %USB_ROOT% > "%USB_ROOT%\openclaw-portable\data\.last_usb"
```

**修改后：**
```batch
echo "%USB_ROOT%" > "%USB_ROOT%\openclaw-portable\data\.last_usb"
```

**原因：** 防止路径包含空格时被截断
