## 📦 2つのバージョン

ニーズに合ったバージョンをお選びください：

| バージョン | サイズ | モデル | 推奨用途 |
|-----------|--------|--------|----------|
| **オフライン版** | ~1.2 GB | ✅ 内蔵 Qwen2.5-1.5B | 完全オフライン使用、設定不要 |
| **オンライン版** | ~300 MB | ❌ なし（自分で追加） | 高速ダウンロード、クラウドAPI使用 |

### オフライン版（推奨）
- ✅ **完全オフライン動作** - ダウンロード後はネットワーク不要
- ✅ **内蔵 Qwen2.5-1.5B モデル** - API費用ゼロ
- ✅ **すぐに使える** - 解凍して実行
- 📥 **ダウンロード**: `OpenClaw-Portable-v6.0.0-windows-offline.tar.gz`

### オンライン版
- ⚡ **サイズが小さい** - ダウンロード速度4倍
- 🌐 **クラウドAPI使用** - DeepSeek、OpenAIなど
- 🤖 **自分のモデルを追加** - カスタムモデル対応
- 📥 **ダウンロード**: `OpenClaw-Portable-v6.0.0-windows-online.tar.gz`

**推奨**: 完全オフライン使用が必要な場合は**オフライン版**を選択してください。安定したネットワークがある場合や特定のクラウドAPIを使用したい場合は**オンライン版**を選択してください。

## 🎭 これらのシナリオに最適

### 🏢 企業環境
- **問題：** ソフトウェアをインストールできない、ネットワークが監視されている
- **解決策：** USBから実行、トレースを残さない
- **結果：** IT承認なしでAIアシスタントを使用可能

### 🌐 制限されたネットワーク（中国、イランなど）
- **問題：** VPNが不安定、ダウンロードが遅い
- **解決策：** すべてがプリインストール、ダウンロード不要
- **結果：** VPNなしで完璧な体験

### ☕ 公共コンピュータ（図書館、カフェ）
- **問題：** インストールできない、データセキュリティの懸念
- **解決策：** ポータブル実行、データはUSB上
- **結果：** 共有デバイスで安全にAIを使用

### 💼 複数のワークステーション
- **問題：** 各マシンで再インストール
- **解決策：** 1つのUSB、すべてのマシンに対応
- **結果：** どこでもシームレスなワークフロー

### 🏔️ リモート/オフライン地域
- **問題：** インターネットなし、AIを使用できない
- **解決策：** 99%オフライン操作
- **結果：** 荒野でもAIアシスタント

## 🚀 クイックスタート（60秒）

### Windows

```bash

# 起動
cd OpenClaw-Portable-v6.0.0
./start.sh

# ブラウザが自動的に開く
```

**これだけ！インストール不要、設定不要、ネットワーク不要。**

## 🔧 高度な使用法

### カスタムポート
`start.bat` を編集して変更：
```batch
set GATEWAY_PORT=18789
```

### カスタム設定
1. `config/openclaw.json.example` を `config/openclaw.json` にコピー
2. 設定ファイルを編集
3. OpenClaw を再起動

### 環境変数（Linux/macOS）
```bash
export GATEWAY_PORT=18790
./start.sh
```

## ❓ よくある質問

### Q: ダウンロードパッケージが这么大（148 MB）なのはなぜ？
**A:** 完全なランタイムを含んでいます：
- Node.js ランタイム（~85 MB）
- OpenClaw + すべての依存関係（~63 MB）
- これが真のオフライン能力の代償です

### Q: 会社のコンピュータで使用できますか？
**A:** はい！すべてのデータはUSB上にあり、ローカルマシンにはゼロトレースです。

### Q: 中国で正常に動作しますか？
**A:** 完璧に！VPN不要、すべてのコンテンツがプリダウンロード済み。

### Q: 起動に60秒以上かかる場合はどうすればいいですか？
**A:** 初回起動はより時間がかかる場合があります。スクリプトは待機を続け、5秒ごとに進捗を表示します。

### Q: トークンはどこで見つけられますか？
**A:** トークンは自動的に抽出され、コンソールに表示されます。ブラウザはトークンを含むURLで自動的に開きます。
