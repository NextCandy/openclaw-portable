# Changelog

All notable changes to OpenClaw Portable will be documented in this file.

## [6.0.0] - 2026-03-15

### Added
- **里程碑版本发布** - 基于 v5.0.5 完全重写
- **新手友好文档** - 面向非技术用户重新设计
- **完善的场景支持** - 嶙新的场景描述
- **智能启动流程** - 60秒等待，进度反馈
- **自动 Token 提取** - 无需手动查找
- **浏览器自动打开** - Token 自动注入 URL
- **零配置运行** - 解压即用，无需配置
- **跨平台支持** - Windows/Linux/macOS 完整支持
- **离线包发布** - 148MB 完整离线包
- **企业级文档** - INSTALL-guide.md, OFFLINE-guide.md
- **多语言支持** - 中/英/日 文档

- **GitHub Release** - 自动发布到 GitHub
- **Vercel 部署** - 支持 Vercel 一键部署
